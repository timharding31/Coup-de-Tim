# Coup-de-Tim
Web-based card game inspired by Rikki Tahta's popular party game, Coup.

## Technologies Used
* JavaScript w/ minimal external libraries
  * Webpack
  * Babel & relevant file loaders
* HTML5
* Sass/CSS
